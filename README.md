Simple Weather App 🌤️

A lightweight, responsive web application that allows users to check real-time weather data for any city worldwide.
🚀 Features

    Search by City: Get instant weather updates by typing the name of a city.

    Real-time Data: Displays Temperature (°C), Wind Speed (km/h), and Rainfall (mm).

    Clean Interface: Simple, focused UI for a quick user experience.

    Responsive Design: Works across different screen sizes.

🛠️ Built With

    HTML5: Semantic structure for the search interface.

    CSS3: Custom styling (linked via style.css).

    JavaScript: Logic for fetching and displaying data (handled in main.js).

    AJAX: Used for asynchronous data retrieval from weather APIs.

📂 Project Structure
Plaintext

├── index.html   # Main entry point and UI structure
├── style.css    # Styling and layout definitions
└── main.js      # Weather fetching logic and DOM manipulation

⚙️ How to Use

    Clone this repository to your local machine.

    Open index.html in your preferred web browser.

    Type a city name (e.g., "Paris", "London", "Tokyo") into the input field.

    Click the "Get Weather" button to view the results.
